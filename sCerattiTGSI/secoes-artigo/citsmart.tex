\section{CITSMART}
\label{sec:citsmart}

 O Citsmart é um software de gestão de TI, auxiliando na tomada de decisões por
 fornecer informações.O Citsmart tem como objetivo manter a eficiência nos processos de prestação de
 serviços e promover sua melhoria, pois é um software aplicativo, desenvolvido para 
 ambiente web, com utilização via intranet e internet, com menus dinâmicos e
 personalizáveis
 A ferramenta de gestão de processos foi criada em 2011 pela equipe da Central IT, como forma de mostrar 
 modelos de gestão rápida e eficaz, de acordo com a biblioteca ITIL
 (\emph{Information Technology Infrastructure Library}) para o Gerenciamento de
 Serviços de TI, sendo sempre atualizada a última foi esse ano, \cite{Citsmart}.
 
 
 O Citsmart possui as seguintes funcionalidades:
 \begin{itemize}
\item Controlar os Ativos, Clientes, Fornecedores e Itens de Configuração
necessários para prover serviços;
\item Promover a Gestão dos Acordos de Nível de Serviço;
\item Elaborar Catálogos e Portfólios de Serviços;
\item Facilitar a elaboração dos Desenhos de Serviço;
\item Facilitar o reconhecimento do Valor do Serviço para o negócio;
\item Prover serviços com qualidade e garantia;
\item Manter um controle de incidentes, problemas e requisições de usuários;
\item Manter o Gerenciamento do Conhecimento de fácil acesso;
\item Gerar relatórios que ajudem a definir planejamentos de mudanças e
melhorias.
 \end{itemize}

 A capacidade técnica rendeu ao Citsmart uma série de certificações PinkVerify, da certificadora de processos ITIL, Pink 
 Elephant, uma garantia de sua eficácia CITSMART \cite{Citsmart}.
 Processos certificados CITSMART \cite{Citsmart}:
 \begin{itemize}
\item Gerenciamento de Incidentes;
\item Cumprimento de Requisições;
\item Gerenciamento de Conhecimento;
\item Gerenciamento de Catálogo de Serviços;
\item Gerenciamento de Níveis de Serviço;
\item Gerenciamento de Mudanças;
 \end{itemize}
 
 CITSMART é um software completo, podendo atender todas as áreas de TI da
 universidade, pois, como citado anteriormente possui varias funcionalidades, 
 como criação de serviços para atender as necessidades de cada organização, podendo 
 também ser personalizado  conforme necessidades especificas das organizações,  por 
 ser um software completo torna-se complexo apresentando assim certas dificuldades 
 para ser trabalhado, por essa razão umas das principais dificuldades seria dar treinamento 
 para que assim fosse usado plenamente.
